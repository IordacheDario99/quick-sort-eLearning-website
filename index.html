Ce este Quick Sort ?

Quick sort este unul dintre cei mai populari algoritmi de sortare avand o complexitate de (O log n) in cazul cel mai favorabi si o complexitate de O(n^2) in cazul cel mai putin favorabil. Quick Sort este unul dintre cei mai buni algoritmi de sortare bazati pe comparatia elementelor atunci cand ni se ofera o multime aleaotrie de elemente, iar numarul elementelor din acea mutlime nu este prea mare



Istorie

Algoritmul de sortare Quick Sort a fost dezvoltat de catre informaticianul de origine britanica, Tony Hoare, in anul 1959. Denumirea de "Quick Sort" vine de la faptul ca, acest algoritm de sortare este capabil sa sorteze o lista de elemente de date semnificativ mai repede (dublu sau chiar triplu in viteza) fata de oricare alt algoritm obisnuit de sortare



















Alti algoritmi de sortare

Cum spuneam, Quick Sort este unul dintre cei mai rapizi algoritmi de sortare, dar asta nu inseamna ca este unic si ca nu avem de unde sa alegem. Un alt algoritm de sortare, care imparte acelasi concept este Merge sort. Principiul de la baza acestor algoritmi de sortare este cel de "Divide Et Impera". Acest principiu presupune descompunerea problemei in doua sau mai multe subprobleme. Principiul divide et impera a fost aplicat de catre imparatul Iulius Cezar
In urmatoarele randuri am sa enumar si alti algoritmi de sortare si dar si cand este cel mai bine ii utilizam.

INSERTION SORT

Este cel mai util atunci cand cele mai multe elemente sunt deja sortate

COUNT SORT

Acest algoritm este util atunci cand în setul de n elemente este dat fiecare interval de elemente

RADIX SORT

Acesta este util cand în setul de n elemente, fiecare element contine numărul de cifre date

SELECTION SORT

Acest algortim, desi este unul dintre cel mai usor de inteles di implementat, el este unul important deoarece de cele mai multe ori este fundamental pentru multi algoritmi de sortare. Acest algoritm este cel mai util atunci cand numarul de mutar solicitat este mai mic.
